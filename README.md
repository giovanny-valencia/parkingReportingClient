# goCite

Report parking violations on the go—and get rewarded!

## Overview

goCite is an Android and iOS mobile app designed to simplify parking violation reporting. Users can create an account, snap photos of violations, add details, and submit reports. Soon, you’ll even earn payments for your contributions with options to withdraw funds. For officers, goCite offers a city map view with pinned reports, showing locations, images, and violation specifics—all built for ease and impact.

## Features

User Accounts: Sign up and manage your profile.
Report Violations: Snap a photo, add details, and submit parking issues.
Map View (Officers): See pinned reports with full details on a city map.
Future Rewards: Earn money for valid reports and withdraw funds (in development).

## Tech Stack

React Native with Expo: Cross-platform mobile app framework.
TypeScript: For type-safe, reliable code.
